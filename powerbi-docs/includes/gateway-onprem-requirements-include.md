## <a name="requirements"></a>Vereisten
**Minimale vereisten**

* .NET 4.6 Framework
* 64-bits versie van Windows 7 of Windows Server 2008 R2 (of hoger)

**Aanbevolen:**

* CPU met acht kernen
* 8 GB geheugen
* 64-bits versie van Windows 2012 R2 (of hoger)
